# CogMaPOI: Cognitive Multi-Agent POI Recommendation System

CogMaPOI is a research-oriented project that proposes a cognitive multi-agent framework for next Point-of-Interest (POI) recommendation. The model incorporates spatio-temporal dynamics, user behavior, and contextual influences to improve recommendation accuracy.

---

## 🚀 Features

* Multi-agent modeling of user behavior
* Cognitive-inspired recommendation mechanism
* Incorporation of spatial and temporal features
* Ablation studies to analyze component contributions

---

## 🧠 Problem Statement

Traditional POI recommendation systems often fail to capture the dynamic and context-aware nature of human decision-making. This project aims to address this limitation by introducing a cognitive multi-agent system that models user interactions more effectively.

---

## 🏗️ Methodology

The proposed approach includes:

* Modeling users as cognitive agents
* Incorporating temporal patterns (time-based behavior)
* Leveraging spatial information (location-based patterns)
* Evaluating the contribution of each component using ablation studies
