The processing methods for the NYC, TKY, and CA datasets strictly follow the descriptions in the paper. Herein, we provide jsonl files converted into an LLM-compatible format, which can be 

directly used for training and testing models such as GPT-4 and Llama 3.1.

The rules for dataset partitioning are as follows:

Test set: Each sample is derived from the last 30 POIs of a user. The first 29 POIs form the input sequence, which is used to predict the ID of the 30th POI.

Training set: Each sample is generated by continuously intercepting 30 POIs, with the interception range being 1 to n-30 (where n is the total number of POIs of a user).
